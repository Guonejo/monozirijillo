# Aprendiendo con Puertas

Aplicación educativa interactiva para niños preescolares.

## Desarrollo

### Instalación

1. Instala Node.js (versión 16 o superior) desde [nodejs.org](https://nodejs.org/)

2. Instala las dependencias:
```bash
npm install
```

### Ejecutar en modo desarrollo

```bash
npm start
```

### Compilar para Windows (.exe)

```bash
npm run build-win
```

El archivo .exe se generará en la carpeta `dist/`.

## Características

- 5 puertas de colores diferentes
- Objetos educativos: Auto, Elefante, Isla, Oso, Unicornio
- Síntesis de voz en español
- Sonidos característicos para cada objeto
- Interfaz amigable para niños preescolares

## Requisitos del Sistema

- Windows 7 o superior
- No requiere conexión a internet
